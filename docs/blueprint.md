# **App Name**: Ternak ID

## Core Features:

- Livestock Profile: Displays detailed information for each of 100 livestock animals including identification, status, and key metrics, dynamically loaded.
- Health Records Management: Allows users to input, view, and manage health records, including vaccinations and treatments, with sortable history.
- Reproduction History: Enables tracking of reproduction events such as insemination and births, with static data display and sortable history.
- Growth Data Analysis: Calculates and displays Average Daily Gain (ADG) based on inputted weight data, with a historical weight table.
- Pedigree Information Input: Provides a form for entering and displaying pedigree information for each animal, supporting genetic improvement programs.
- Dynamic Identity Editing: A modal form that allow user to edit basic livestock information: birthdate, owner name, address, name, breed, gender and status.
- Livestock Health Prediction: Predict the likelihood of livestock developing certain health issues based on historical health records. LLM analyzes past health events to anticipate potential future problems, alerting farmers to take preventive measures.

## Style Guidelines:

- Primary color: Emerald green (#34D399) to reinforce the natural, agricultural themes.
- Background color: Light gray (#F9FAFA) to create a clean and modern appearance.
- Accent color: Sky blue (#38BDF8) used sparingly for interactive elements.
- Body and headline font: 'Inter', a sans-serif font that is modern, clear and functional
- Use consistent and clear icons from a set like Font Awesome, related to livestock, health, and growth tracking.
- The main layout is divided into a left sidebar navigation with a content area on the right, with clear tabs to separate types of livestock information.
- Subtle transitions and loading animations to improve user experience, such as when switching between tabs or saving data.
